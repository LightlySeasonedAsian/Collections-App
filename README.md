# Collections App
## Setup Local Environment_

- Pull Repo into local machine
- BASH console run this command: npm run setup-dev
```sh
//pulls dependencies
npm run setup-dev
//starts the application, and watches for file changes
nodemon index.js 
```

## Installation

App requires [Node.js](https://nodejs.org/) v16+ to run.
